# 네이버 재테크 카페 시딩 본문

네이버 카페는 한국 검색 트래픽의 가장 큰 backlink 자산. 단 **가입 즉시 글쓰기 금지** — 1주일 평판 빌드 후 발사.

## 추천 카페 (회원 수 + 미주 분석 친화도 기준)

| 카페 | 회원 수 | 미주 분석 친화도 | 등급 요건 | 비고 |
|------|--------|---------------|----------|------|
| **재테크인사이드** | 100만+ | ★★★★★ | 가입 후 7일 + 댓글 30개 | 가장 큼, 분석글 환영 |
| **한경 미국주식 카페** | 50만+ | ★★★★ | 가입 후 14일 + 출석 5일 | 진지함 |
| **미주미주** | 30만+ | ★★★ | 즉시 가능 | 캐주얼, 진입 쉬움 |
| **실전 미국주식** | 20만+ | ★★★ | 가입 후 3일 | 중간 |

## 발사 전 필수 단계 (1주일 평판 빌드)

1. **가입** → 프로필 사진 + 한 줄 소개 작성 ("미국주식 분석 + 우주산업 관심" 정도)
2. **출석** 매일 1회 (5일 이상)
3. **다른 사람 글 추천** 50개+
4. **댓글** 30개+ (단순 "공감" X — "저도 이 생각이었는데 ~~점은 어떻게 보세요?" 식 진지한 댓글)
5. **카페 매니저/스탭 글에 댓글로 인사** (평판 가속 트릭)

위 5개 끝나면 글쓰기 권한 + 운영자 호감 둘 다 확보됨.

## 본문 1 — SATS 분석 (재테크인사이드 / 실전 미국주식 추천)

### 제목
```
[분석] EchoStar(SATS) — 이미 9배 오른 SpaceX 백도어, 지금 들어가도 될까
```

### 본문

```
안녕하세요. 며칠 전 EchoStar(SATS) 차트 보다가 9배 오른 게 눈에 띄어서
재무제표까지 파보고 정리해봤습니다. 결론부터 말씀드리면, 이미 익은 트레이드라
지금 신규 진입은 신중히, 기존 보유자는 일부 익절 + 일부 holding 이 합리적
이라고 봅니다. 이유 정리해드립니다.

▶ 무슨 일이 있었나
2025년 9월 8일, EchoStar 가 보유한 AWS-4 (2GHz) + H-block (1.9~2.0GHz)
주파수를 SpaceX 에 약 $17B 에 매각. 2주 후 AWS-3 비페어드 추가 매각
$2.6B. 합쳐서 $19.6B 거래.

지급 구조 (이게 핵심):
- $8.5B 현금 + $8.5B SpaceX 비공개 주식 (1차)
- $2B SpaceX 의 EchoStar 이자 대납 (2027.11까지)
- $2.6B SpaceX 비공개 주식 (2차, AWS-3)

즉 EchoStar 는 약 $11.1B 어치의 SpaceX 비공개 주식 + $8B 현금을 받았고,
$25B 부채는 $15B 로 감소.

▶ 왜 SpaceX 가 $19.6B 을 썼나
AWS-4/H-block/AWS-3 은 위성-스마트폰 직접 통신(D2C) 에 쓸 수 있는 가장
깨끗한 미드밴드 주파수. 기존 T-Mobile + Starlink 협력은 T-Mobile 의 지상망
주파수를 빌려 쓰는 구조였는데, 이걸 SpaceX 가 자기 주파수로 쓸 수 있게 됨.
즉 T-Mobile 의존을 끊는 transaction 이고, $19.6B 은 합리적 가격.

▶ 거래 후 SATS 차대변
- 부채: $25B → $15B
- 주파수: $23B → $3B (잔여)
- SpaceX 비공개주식: $0 → $11.1B
- 현금: $2B → $8B
- 잔여 사업: Boost (752만 가입자, 유일 성장) + Hughes (스타링크에 잠식) +
  Dish (코드커팅 진행)

현재 시총 ~$30B 기준으로, SpaceX 지분 + 현금 빼면 잔여 통신사 + 옵셔널리티
가 ~$11B 으로 평가받는 셈. 합리적인 수준.

▶ 강세 시나리오 3가지
1. SpaceX IPO 가 $1.75T 컨센서스로 클리어 → SATS 보유 SpaceX 주식이
   3~4배 마크업. 자동으로 발생.
2. Boost × Starlink D2C 통합으로 ARPU 상승
3. AT&T/Verizon 인수 옵셔널리티 (Ergen 73세, M&A 가능성 살아남)

▶ 약세 시나리오 4가지
1. $15B 잔여부채 — 2027.11 이자 브릿지 종료 후 자체 부담
2. SpaceX 비공개주식의 비유동성 — IPO 시점·lockup 따라 12~24개월 미뤄질
   가능성
3. Charlie Ergen 의 ~50% 의결권 → 2017년 $40 에 안 판 사람
4. **이미 9배 가격 반영** — 다음 50% 는 양성 서프라이즈 필요

▶ 제 결론
- 신규 진입: 어려움. 풀백 ($80~$95) 기다리는 게 합리적
- 기존 보유: 30~50% 익절 + 나머지 IPO 까지 holding
- 신규 SpaceX 노출이 목표면: XOVR 이 더 깨끗 (NAV 프리미엄 없음, 분산됨)

SATS 의 "유효 SpaceX 노출" 은 ~37% 로 미국 유동 비히클 중 최고. 단 부채와
사라지는 통신사가 함께 묶인 wrapped exposure 라는 점 잊지 말 것.

차대변 + 강세/약세 + 사이징 룰 좀 더 자세히 정리해놓은 글 출처 남겨드립니다:
https://luckyplz.com/blog/echostar-sats-spacex-backdoor/

다른 분 의견도 듣고 싶습니다. SATS 어떻게 보고 계신가요?
```

## 본문 2 — ETF 비교 (재테크인사이드 추천)

### 제목
```
[정보] SpaceX 노출 ETF 3종 정밀 비교 — XOVR vs DXYZ vs RONB ($1당 효율)
```

### 본문

```
SpaceX IPO 다가오면서 ETF 로 노출 가지려는 분들 많은데, 3종 비교 정리한 거
공유드립니다. 결론은 대부분에게 XOVR 추천이고, DXYZ 는 함정 있어요.

▶ 한 줄 비교

| ETF | SpaceX 비중 | NAV 프리미엄 | 운용보수 | 추천도 |
|-----|-----------|------------|---------|--------|
| XOVR | ~16% | ≈0% | 0.75% | ★★★★★ |
| DXYZ | ~23% (명목) | +30~120% (위험) | 2.5% | ★★ |
| RONB | 14~22% | -10~+10% | 1.0% | ★★★ |

▶ DXYZ 의 함정 (가장 주의)
명목 SpaceX 비중은 가장 높지만, 거래가가 NAV 의 +30~120% 프리미엄.
이게 closed-end fund 특성인데, IPO 이벤트 시점에 두 가지 일이 동시에 발생:
1. NAV 가 점프 (보유 SpaceX 주식이 상장가로 마크업)
2. 프리미엄이 붕괴 (closed-end fund 가 더 이상 'SpaceX 못 사는 사람의
   유일한 통로' 가 아니게 됨)

(1) 의 업사이드를 (2) 가 깎아먹어서, 명목 23% 의 절반 정도 노출만 실제로
받게 됨. 즉 "$1당 SpaceX 노출 효율" 기준으로:

XOVR > RONB > DXYZ

▶ XOVR 가 베이스인 이유
- SpaceX 비중 16% — 의미있게 높지만 SpaceX 단일 베팅 X
- 다른 보유: OpenAI/Stripe/Anthropic — 진짜 분산 (DXYZ 는 100여종 얇게)
- NAV 프리미엄 없음 — 들어가는 가격 = 자산 가격
- 유동성 양호 + 운용보수 합리적

단점: SpaceX 가 +100% 가도 ETF 는 +16% 만 반영 (SpaceX 단일 베팅이면 만족도
낮을 수 있음). 그게 싫으면 SATS (~37% 효과적 노출) 가 답.

▶ RONB
신생이고 유동성 얇음. 단 운용보수 1.0% 로 DXYZ 보다 저렴, NAV 프리미엄도
없거나 약하게 +/-. 얼리 어답터에게 OK.

▶ 한국에서 매수
키움 영웅문Global / 미래에셋 m.STOCK / 토스증권 / 삼성증권 m.POP 모두 가능.
세금: 양도세 22% (250만원 기본공제), 환차익 비과세.

▶ 3가지 포트폴리오 템플릿
1. 보수형: 미주식 95% + XOVR 5%
2. 밸런스형: 미주식 88% + XOVR 5% + DXYZ 5% + 현금 2%
3. 공격형: 미주식 80% + XOVR 5% + RONB 3% + SATS 2% + 현금 10%

세금·NAV 프리미엄 동작·portfolio 템플릿 자세히 정리:
https://luckyplz.com/blog/spacex-etf-comparison/

다른 분들 어떤 ETF 비중으로 가져가고 계신가요?
```

## 본문 3 — Musk 자산 (미주미주 / 캐주얼 카페 추천)

### 제목
```
[잡담] 머스크가 SpaceX 상장 후 첫 트릴리어네어가 될 수 있을까 — 6개 회사 모델링
```

### 본문

```
요새 머스크 자산 얘기 많이 나오는데, 직접 모델링해봤어요. 결론: 베이스 케이스
$940B, 풀 시나리오로는 $1.4T. 인류 첫 트릴리어네어 가능성 매우 높아 보입니다.

▶ 6개 회사 지분 합산

1. SpaceX 42% × $1.75T = $735B (메인)
2. Tesla 13% × $1.2T = $155B
3. X (옛 트위터) 80% × $50B = $40B
4. xAI = SpaceX 합병으로 흡수 (별도 X)
5. Boring Company ~80% × $7B = $5.5B
6. Neuralink ~50% × $9B = $4.5B

합계: ~$940B (약 1,300조 원)

▶ 시나리오별
- Bear (SpaceX $1.2T IPO): $700B
- Base (SpaceX $1.75T IPO): $940B
- Bull (SpaceX $2.0T+): $1.2T+

▶ 포브스 vs 블룸버그 (이게 흥미로운 부분)
같은 사람 자산을 두 매체가 $300B 차이로 발표하는 이유:
- 포브스: 최근 거래가 (private 마지막 round 평가 그대로)
- 블룸버그: 실시간 mark-to-market + 디스카운트 적용

포브스 기준이면 IPO 당일 $1T 돌파 가능, 블룸버그 기준이면 2027년쯤 돌파.
"누가 첫 트릴리어네어냐" 는 결국 어느 매체를 보느냐 문제.

▶ 중요 caveat
순자산 ≠ 유동성. 대부분이 lockup + insider trading 제약 + 담보 대출에 묶임.
실제로 머스크가 매도 가능한 부분은 매우 제한적 (Tesla SOX 11 plan 따른
계획 매도 정도).

▶ 한국 투자자 입장에서 시사점
머스크가 트릴리어네어 되는 시나리오는 결국 SpaceX 가 $1.75T+ 로 클리어하는
경우. 이게 깨지면 SpaceX 노출 자산 (XOVR/SATS/Tesla) 전체가 같이 빠짐.
즉 "트릴리어네어 베팅 = SpaceX 베팅" 으로 봐야 함.

스테이크별 모델 + Bull/Base/Bear 정리: https://luckyplz.com/blog/musk-net-worth-after-spacex-ipo/

다들 머스크 자산 어떻게 보고 계세요?
```
