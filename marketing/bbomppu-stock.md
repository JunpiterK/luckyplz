```
NAV 프리미엄 없고, OpenAI/Stripe/Anthropic 같은 분산도 됨.

[방법 2] EchoStar (SATS) — 백도어

$25B → $15B 부채 감소 + ~$8B 현금 + ~$11B SpaceX 주식 보유 상태로 변환.

SATS 시총의 약 37% 가 SpaceX 노출. 미국 유동 비히클 중 최고 농도.

- 주가가 이미 $14.90 → $137 로 9배 오름 (이미 가격에 반영)

- Charlie Ergen 의 ~50% 의결권 → 과거 합리적 오퍼 거절 이력

신규 진입은 풀백 ($80~$95) 기다리는 게 합리적. 기존 보유자는 일부 익절 +
일부 holding.

------------------------------------------------------------

단점: IPO 첫날 +30~50% 프리미엄 가능성. 일주일~한 달 기다리는 게 합리적.
```

## 본문 2 — 주식 게시판 / 짧은 톤

### 제목
```
[공유] SATS 9배 오른 후, 추가매수 vs 익절 — 차대변 분석
```

### 본문

```
EchoStar (SATS) 가 작년 9월 $14.90 에서 지금 $137 까지 약 9배 올라서,
보유자분들 추가매수 vs 익절 고민 많을 것 같아 차대변 분석해봤습니다.

▶ 거래 후 SATS 의 모습
- 부채: $25B → $15B (-$10B)
- SpaceX 비공개주식: $0 → $11.1B (NEW)
- 현금: $2B → $8B
- 주파수: $23B → $3B (잔여)
- 운영사업: Hughes (쇠퇴) + Boost (752만 가입자, 성장) + Dish (쇠퇴)

▶ 시총 분해
현재 ~$30B 시총 = SpaceX 지분 $11B + 현금 $8B + 잔여 사업 $11B

즉 시장이 "잔여 통신사 + 옵셔널리티" 에 $11B 가치 매기는 셈. Boost 만으로는
좀 비싸지만, M&A 옵셔널리티 + Boost × Starlink D2C 통합 + SpaceX 마크업
가능성을 합치면 합리적.

▶ 강세 시나리오 (자동 발생)
SpaceX 가 IPO 시 $1.75T 컨센서스 기준이면, SATS 보유 SpaceX 주식이
3~4배 마크업 → SATS 장부 가치 +$30~$40B 추가.

이건 SATS 운영진 노력 0이어도 발생. SpaceX IPO 가 컨센서스대로만 되면.

▶ 약세 시나리오
1. $15B 부채는 2027.11 SpaceX 이자 브릿지 종료 후 자체 부담
2. SpaceX 주식이 비유동적 — IPO 후에도 lockup 6~12개월
3. Charlie Ergen 의결권 50%+ — 과거 $40 에 안 판 사람
4. **이미 9배** — 다음 50% 는 양성 서프라이즈 필요

▶ 결론
- 신규 진입: 어려움. $80~$95 풀백 기다림이 합리적
- 기존 보유: 30~50% 익절 + 나머지 IPO 까지 holding (라운드트립 방지)
- 순수 SpaceX 노출 목표면: XOVR 가 더 깨끗

▶ 사이징 룰
SATS = SpaceX 버킷의 최대 15%. 더 늘리면 잔여 통신사 리스크가 SpaceX
업사이드를 압도.

차대변 + 강세/약세 + 사이징 자세히: https://luckyplz.com/blog/echostar-sats-spacex-backdoor/

다들 어떻게 가져가고 계세요?
```
