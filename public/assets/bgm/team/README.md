# 👥 Team (팀뽑기) BGM

캡슐 가챠 머신. 통통 튀는 리듬.

## Suno 프롬프트

### 옵션 A — 가챠 펑키
```
playful 8-bit chiptune gacha machine theme, bouncy and energetic,
130 BPM, syncopated square bass, snappy chip drums, cute Kirby-style
melody, instrumental, loopable, no vocals
```

### 옵션 B — 공장 라인 리듬
```
cheerful 8-bit factory conveyor music NES-era, 125 BPM, repetitive
bouncy 8-bar loop, square lead and triangle bass, light percussion,
mood between Tetris menu and Animal Crossing shop, instrumental,
no vocals
```

### 옵션 C — 캡슐 등장
```
upbeat 8-bit toy machine chiptune, 135 BPM, staccato square melody,
syncopated triangle bass with octave jumps, snappy chip kick, fun
Mario-Kart-menu vibe, instrumental, loopable, no vocals
```

## 트랙 배치

```
public/assets/bgm/team/
├── track1.mp3
├── track2.mp3
└── track3.mp3   (선택)
```

권장: 옵션 A 1곡 + B 1곡. 팀뽑기는 한 라운드 1~2분이라 60~90초 트랙으로 적당.

## Exclude
```
vocals, lyrics, EDM, dubstep, lo-fi
```
