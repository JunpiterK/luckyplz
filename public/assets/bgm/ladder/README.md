# 🪜 Ladder (사다리) BGM

세로 낙하의 미스터리한 분위기. 결과 공개 전 두근거림.

## Suno 프롬프트

### 옵션 A — 미스터리 chiptune
```
mysterious 8-bit chiptune with a falling melodic descent, Game Boy era
exploration vibe, 110 BPM, minor key, plucky square lead, sparse
arpeggio, slight dreamy reverb, instrumental, loopable, no vocals
```

### 옵션 B — 동화 같은 cute
```
gentle storybook chiptune NES-style, descending arpeggios, 100 BPM,
soft and curious mood like Earthbound town theme, square lead with
triangle bass, instrumental, no vocals, loopable
```

### 옵션 C — 가벼운 puzzle
```
calm puzzle-game chiptune in C minor, 105 BPM, repetitive 16-bar loop,
square lead playing simple descending phrases, triangle bass holding
quarter notes, light hi-hat, NES era, instrumental, no vocals
```
