# 🎯 Roulette (룰렛) BGM

원형 스피너 회전 → 결정의 순간. 카지노 + 귀여움 조합.

## Suno 프롬프트

### 옵션 A — 카지노 8-bit
```
8-bit chiptune NES-style casino spinning wheel music, playful suspense
building to a cheerful resolve, 120 BPM, square wave melody, triangle
bass, hi-hat, instrumental, loopable, retro game soundtrack, light
tension, no vocals
```

### 옵션 B — 가벼운 펀
```
upbeat retro arcade chiptune for a roulette game, bouncy and cute
Pokemon-style overworld vibe, 130 BPM, simple repetitive 4-bar loop,
square lead, soft chip drums, no vocals, instrumental
```

### 옵션 C — 가족적인 게임
```
cheerful 8-bit chiptune game-show wheel theme, anticipation building,
115 BPM, bright square lead, triangle bass walking line, light chip
percussion, Mario-Party-style upbeat mood, instrumental, loopable,
no vocals
```

## 트랙 배치


## Exclude
```
vocals, lyrics, EDM, dubstep, modern trap, lo-fi hip hop
```
