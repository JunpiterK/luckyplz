공이 추출되는 물리 시뮬. 기대감 → 환호.

## Suno 프롬프트

### 옵션 A — 추첨 빌드업
```
8-bit chiptune lottery drawing music, anticipation building, 100 BPM,
slow swelling square pad, light triangle bass, sparse percussion,
classic NES game show feel, instrumental, loopable with subtle
drumroll texture, no vocals
```

```
calm and curious 8-bit chiptune, slow tempo 90 BPM, like Pokemon
Center theme but for a number-drawing machine, soft square arpeggio,
gentle triangle bass, mysterious but warm, instrumental, loopable,
no vocals
```

### 옵션 C — 부드러운 cinematic
```
gentle 8-bit cinematic chiptune for a number drawing machine, 95 BPM,
soft minor-to-major resolution feel, slow square arpeggio, sparse
triangle bass, subtle white noise wash, dreamy and patient mood,
instrumental, loopable, no vocals
```
