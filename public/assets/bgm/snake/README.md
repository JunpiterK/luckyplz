# 🐍 Snake BGM

고전 아케이드. 차분 + 집중.

## Suno 프롬프트

### 옵션 A — 클래식 calm arcade
```
calm minimalist 8-bit chiptune for a snake arcade game, 100 BPM,
simple repetitive square melody, sparse triangle bass, no drums,
focused puzzle-game vibe like Game Boy Tetris B-theme, instrumental,
loopable, no vocals
```

### 옵션 B — 살짝 mysterious
```
mellow 8-bit chiptune in minor key, 95 BPM, sparse but hypnotic
square arpeggio, light triangle bass, calm and meditative arcade
mood, NES-era, instrumental, loopable, no vocals
```

### 옵션 C — 점진적 빌드업
```
8-bit chiptune for a slowly-intensifying arcade game, 105 BPM,
minimalist square lead, walking triangle bass, gradually adding
hi-hat after 8 bars, focused gameplay vibe, instrumental, loopable,
no vocals
```
